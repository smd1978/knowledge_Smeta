Создаем репозитарий Git

    1. git init //инициализируем git в папке проекта
    2. git config --local user.name "Mihail" //настраиваем систему Git
    3. git config --local user.email "smd78@list.ru" //настраиваем систему Git
    4. git status //Проверяем состояние файлов
    5. git add [имя файла] или git add -A (все файлы) //Делаем все файлы отслеживаемыми 6.
